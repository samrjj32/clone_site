# MeetingNotesSection Specification

## Overview
- **Target file:** `src/components/MeetingNotesSection.tsx`
- **Interaction model:** static
- **Purpose:** Showcase Cluely's meeting notes feature with a large screenshot

## DOM Structure
```
<section> (mx-auto flex w-full max-w-7xl flex-col items-center gap-10 px-5 md:gap-14 md:px-8)
  <div class="text-center">
    <h2> "Instant meeting notes"
    <p> "The easiest way to get beautiful, shareable meeting notes."

  </div>
</section>
```

- **Desktop (1440px):** heading ~44px, screenshot full width of container
